# HTML5 drag and drop API 
Example of HTML5 drag and drop API

### HTML5 Drag and Drop  
Drag and Drop 기능은 사용자들에게 직관적인 UI를 제공할 수 있다.  
예) 파일 업로드, 레이아웃 변경, 아이템 선택 후 타겟 위치에 이동  
HTML5 DnD는 DOM event model과 마우스 이벤트에서 상속된 drag event를 사용한다.
